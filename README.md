Models for contrastive reinforcement learning.

## Open AI Gym

To install open AI gym (on Mac):

```bash
git clone https://github.com/openai/gym.git
cd gym
pip install -e .
pip install pyglet==1.5.11
```

You may need to install `ffmpeg`

```bash
brew install ffmpeg
```
